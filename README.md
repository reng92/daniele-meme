# I Meme Daniele

App Next.js che genera meme divertenti in italiano usando Groq AI.

## Avvio locale

1. Installa le dipendenze:

   npm install

2. Crea `.env.local` partendo da `.env.example` e inserisci la tua chiave Groq.

3. Avvia:

   npm run dev

## Variabili ambiente

- `GROQ_API_KEY`
- `NEXT_PUBLIC_BASE_URL`

## Deploy

Deploy consigliato su Vercel collegando questa repository.
